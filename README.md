# TestNxViteTreeShaking

This is a PoC to demonstrate tree shaking behaviour on Nx Monorepo with a Vite app (React + Typescript)

## Build the project

Use the Nx build command: 
```bash
npx nx build frontend --skip-nx-cache --emptyOutDir
```
